# Design Direction

## Surface
Internal analytics dashboard for a 27-inch desktop monitor, with responsive support for laptop and mobile checks. Users are likely reviewing during planning and readout prep, so the interface should be bright, calm, and scan-friendly.

## Visual System
- Light theme with warm off-white background and ink-tinted neutrals.
- Use brand-coded data colors consistently: Shapermint purple, Spanx amber, Skims pink, Maidenform blue, Honeylove red.
- Use restrained but confident accents. The interface should feel editorial and decisive, not decorative.
- Avoid nested cards. Use full-width bands, compact panels, tables, and chart areas.

## Interaction
- Tabs for Awareness, Equity Ownership, Channels, and Data Notes.
- Metric toggles for aided vs. unaided awareness.
- Hover/focus tooltips on chart points and heatmap cells.
- Compact source badges that distinguish verified, deck-derived, raw export, and missing values.

## Typography
Use system sans-serif. Keep headings tight and chart labels readable. Avoid oversized hero text inside dashboard panels.
